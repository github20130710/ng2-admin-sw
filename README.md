# ng2-admin-sw
ng2 admin template for practise

项目使用npm做为包管理器 gulp进行编译,所以请先在环境中安装npm和gulp

项目下载后,请执行 gulp install, 安装项目所需要的包

编译项目请执行 gulp build

启动项目请执行 gulp serve


# 文件结构说明

assets包含了全部的资源文件

theme是主题文件夹,包括头部 尾部 菜单栏等

styles包含了一整套的样式文件,使用less编写

scripts中包含了filter service directive util等等

vendors是所有的插件集合

pages是内容页面集合
